# Java Dice Roller

A simple **Java console application** that simulates rolling one or more dice. Designed for beginners to practice **Java basics**, **loops**, **methods**, and **random number generation**.

---

## Features

- Roll any number of dice
- Displays the result of each die roll
- Shows ASCII art representation of each die
- Calculates and displays the total of all dice rolled
- Prevents invalid input (number of dice must be > 0)
